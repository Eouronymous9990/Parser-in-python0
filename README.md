Parser Generator - LL(1) & SLR(1)
A Python tool for generating LL(1) and SLR(1) parsing tables from context-free grammars.
Features

LL(1) Parser Generator: Creates predictive parsing tables for LL(1) grammars
SLR(1) Parser Generator: Builds Simple LR parsing tables with ACTION and GOTO tables
FIRST & FOLLOW Sets: Automatically computes FIRST and FOLLOW sets for all non-terminals
Conflict Detection: Identifies and reports grammar conflicts
Formatted Output: Clean, readable tables with proper alignment
